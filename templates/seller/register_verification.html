Verification Infomation

You must verify the app ownership before proceeding

1 Add a unique token to your app's description
Copy the token below and paste it into the description field of your app's App Store profile. If you're not sure how to do this, Apple's official documentation describes how to edit your app's description and other meta data here: Apple Documentation: Editing Metadata and Uploads You can place the token anywhere in the description but we suggest you put it at the end of the description so it's not immediately visibile to users.
2 Take action to verify the token's presence
Once you've added the token to your app's App Store page, click the button below and we'll verify the token's presence. That's all you have to do! The verification process can take up to 3 hours In the meanwhile we wait for Apple to update the your app's App Store description. We will notify you via email once we've verified the token after that you can remove it.
